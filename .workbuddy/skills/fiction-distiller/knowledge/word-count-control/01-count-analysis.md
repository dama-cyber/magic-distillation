# 01-count-analysis：原文字数结构分析

## 1. 分析目标

原文字数结构分析是短篇换元仿写的第一步。通过分析原文的字数、段落、句子、对话等结构，为后续生成新篇提供精确的"字数预算表"。

## 2. 分析维度

### 2.1 总字数

- 统计原文不含标点/含标点的字数
- 确定新篇目标字数范围（±5%）

### 2.2 段落结构

- 段落数
- 每段字数
- 每段功能（开头、发展、转折、结尾等）
- 段落长度分布

### 2.3 句子结构

- 句子总数
- 每句字数
- 句长分布（短/中/长句比例）
- 句长标准差 / 均值（突发性）

### 2.4 内容类型占比

- 对话字数占比
- 叙述字数占比
- 描写字数占比
- 心理活动字数占比

### 2.5 节奏分布

- 紧张段落字数
- 舒缓段落字数
- 紧张舒缓比
- 节奏转折点位置

## 3. 统计方法

### 3.1 中文分句规则

- 以句号、问号、感叹号、省略号结尾为一句
- 引号内的对话视为独立句子或段落内的子单元
- 分号、冒号后的内容根据语境判断是否独立成句

### 3.2 字数统计规则

- 中文字符：每个汉字算 1 字
- 标点符号：可选是否计入
- 英文字母：通常按单词或字符统计，根据需求决定
- 数字：每个数字算 1 字

### 3.3 对话识别规则

- 以引号包裹的内容为对话
- 对话提示语（"他说""她回答"）计入叙述部分
- 对话中的动作描写计入描写部分

## 5. 分析工具

`word_count_analyzer.py` 的核心逻辑：

```python
import re

def analyze(text):
    # 分句
    sentences = re.split(r'[。！？…]+', text)
    # 分段
    paragraphs = text.split('\n\n')
    # 统计
    total_chars = len(text)
    sentence_lengths = [len(s) for s in sentences if s]
    paragraph_lengths = [len(p) for p in paragraphs if p]
    # 对话
    dialogue = re.findall(r'"([^"]*)"', text)
    dialogue_chars = sum(len(d) for d in dialogue)
    # 输出
    return {
        "total": total_chars,
        "paragraphs": len(paragraphs),
        "sentences": len(sentences),
        "sentence_lengths": sentence_lengths,
        "paragraph_lengths": paragraph_lengths,
        "dialogue_ratio": dialogue_chars / total_chars
    }
```

## 6. 常见短篇字数结构模式

| 类型 | 总字数 | 段落数 | 平均段落字数 | 对话占比 | 特点 |
|------|--------|--------|--------------|----------|------|
| 微型小说 | 300-800 | 3-5 | 100-200 | 20-40% | 极简，结尾反转 |
| 短篇故事 | 2000-5000 | 8-15 | 200-400 | 25-35% | 完整起承转合 |
| 文学短篇 | 3000-8000 | 10-20 | 300-500 | 15-25% | 描写多，节奏慢 |
| 悬疑短篇 | 2000-4000 | 8-12 | 250-400 | 30-40% | 对话推动，节奏快 |

## 7. 分析注意事项

- 排除标题、作者信息、注释等非正文内容
- 统一标点统计规则（是否计入字数）
- 对话内容单独统计，便于后续控制
- 特殊排版（如空行、分段）需要规范化
- 如果原文分段不规范，需要先进行段落重组

## 8. 输出：字数预算基础

- 原文结构分析报告
- 字数预算表初稿
- 每段目标字数
- 内容类型目标占比
- 节奏目标分布
