# 03-rhythm-perturbation：节奏扰动方法

## 1. 为什么需要节奏扰动

AI 生成文本通常具有均匀的句长和节奏，这是检测工具识别 AI 的重要依据之一。**突发性（Burstiness）** 指标就是衡量这种不均匀程度的。

节奏扰动是主动向文本中引入不均匀的节奏，让文本更像人类写作。

## 2. 节奏扰动的核心指标

### 2.1 句长分布

| 句型 | 字数范围 | 人类写作特征 |
|------|----------|--------------|
| 超短句 | 2-8 字 | 用于停顿、强调、情绪爆发 |
| 短句 | 9-15 字 | 推进节奏，制造紧张 |
| 中句 | 16-30 字 | 主要叙述单位 |
| 长句 | 31-50 字 | 描写、心理、复杂动作 |
| 超长句 | 50字+ | 意识流、紧张堆叠 |

**目标**：句长标准差 / 均值 > 0.7。

### 2.2 段落长度分布

人类写作不会让所有段落长度相近。段落长度应该有明显波动。

**AI 式段落分布**：
> 150, 160, 155, 165, 150...（过于均匀）

**人味化段落分布**：
> 80, 220, 150, 300, 90, 180...（有波动）

**目标**：段落长度有明显的峰谷。

### 2.3 紧张舒缓比

文本应该在紧张和舒缓之间交替，而不是一直紧张或一直舒缓。

**目标**：紧张与舒缓段落比例接近原文，但允许局部波动。

## 3. 节奏扰动技术

### 3.1 超短句插入

在紧张或情绪高潮处插入 2-8 字短句。

**示例**：
> 他推开门。屋里很暗。没人。桌上有一封信。很薄。

**技巧**：
- 在动作转折点插入短句
- 在情绪爆发处用短句加强
- 短句数量占全文 20%-30%

### 3.2 长句堆叠

在需要复杂描写或心理活动时，使用 40 字+ 长句。

**示例**：
> 他站在门口，看着那封信，想起三年前的那个夜晚，也是这样的雨天，也是这样一个没有署名的人，也是这样一句话，让他从此无法安宁。

**技巧**：
- 长句用于回忆、心理、环境渲染
- 长句后接短句，形成节奏落差
- 不要连续使用多个长句

### 3.3 段落长度不均

刻意让段落长度有起伏，避免机械化。

**示例**：
- 第 1 段：80 字（悬念开头）
- 第 2 段：220 字（人物出场）
- 第 3 段：150 字（过渡）
- 第 4 段：300 字（冲突爆发）
- 第 5 段：90 字（情绪转折）

**技巧**：
- 紧张段落可以长
- 留白段落可以短
- 转折处用短段落制造停顿

### 3.4 突然停顿

在紧张推进中突然插入一个停顿或环境描写。

**示例**：
> 他追了上去。脚步声在巷子里回响。拐弯。前面没路了。他喘着气。墙角的猫看了他一眼，又低下头舔爪子。

**技巧**：
- 用无关细节打断紧张节奏
- 用环境描写放缓情绪
- 停顿后再次加速

### 3.5 对话节奏变化

让对话长度不均匀，避免每句话都标准长度。

**AI 式对话**：
> "你来了。""我来了。""你为什么来？""因为我想见你。"

**人味化对话**：
> "你来了。""……嗯。""为什么？""不知道。就是想。"

**技巧**：
- 有问无答
- 有答非所问
- 使用省略号
- 使用短词和停顿

### 3.6 标点运用

通过标点制造节奏变化。

| 标点 | 用法 | 效果 |
|------|------|------|
| 句号 | 提前断句 | 制造停顿 |
| 逗号 | 增加句内停顿 | 放缓节奏 |
| 省略号 | 留白、犹豫 | 情绪悬停 |
| 破折号 | 插入、打断 | 节奏跳跃 |
| 感叹号 | 情绪爆发 | 加强语气 |
| 问号 | 质疑、不确定 | 制造张力 |

**技巧**：
- 模仿原文的标点习惯
- 在紧张处多用句号和短句
- 在留白处多用省略号
- 在转折处多用破折号

### 3.7 句子开头多样化

避免每句都以相同方式开头。

**AI 式开头**：
> 他…他…他…（连续主语开头）

**人味化开头**：
> 他站在门口。风很大。门没有关。屋里有一股旧木头味。他深吸一口气。

**技巧**：
- 用环境开头
- 用感官开头
- 用动作开头
- 用时间开头
- 偶尔省略主语

## 4. 节奏扰动与字数对齐的关系

节奏扰动会影响字数，因此需要在字数控制框架内进行：

- 短句插入不显著增加字数（可能减少）
- 长句堆叠可能增加字数，需在其他地方压缩
- 段落长度不均仍在总字数预算内
- 字数修正阶段要保留节奏扰动成果

## 5. 节奏扰动检查清单

- [ ] 句长分布有明显峰谷（标准差/均值 > 0.7）
- [ ] 有 2-8 字超短句
- [ ] 有 40 字+ 长句
- [ ] 段落长度不均匀
- [ ] 紧张与舒缓有交替
- [ ] 对话长度有变化
- [ ] 标点运用多样
- [ ] 句子开头多样化
- [ ] 节奏变化自然，不刻意
- [ ] 没有因节奏扰动破坏字数目标

## 6. 节奏扰动 Prompt 模板

在生成 prompt 中加入：

```
节奏要求：
- 句子长短交错，短句占比约 30%
- 允许出现 2-5 字超短句
- 允许出现 40 字+ 长句
- 段落长度不要均匀，可以有 80 字短段和 300 字长段
- 紧张处句子密集，舒缓处句子稀疏
- 使用句号、省略号、破折号制造停顿
- 避免连续三句以上长度相近
```
