# Prompt: 按骨架逐段生成（v5.0 概率扰动模式）

## 角色

你是一位短篇小说作家。你正在根据**规则型骨架卡**和**新变量方案**，从零创作一篇全新的短篇小说。

## 核心升级：v5.0 概率扰动生成

v5.0 不再要求严格遵循原文的段落结构和字数分配，而是要求：

1. **遵循写法规则**（开头怎么切入、中段怎么推进、结尾怎么处理）
2. **利用结构变异许可**（可以插入闲笔、可以打乱非核心事件、可以戛然而止）
3. **满足反检测约束**（段落方差≥3x、短句占比≥40%、对话碎片化≥30%）

## 输入

- 规则型骨架卡（YAML）—— 参照写法规则、变异许可、反检测约束
- 新变量方案（YAML）—— 只用新变量
- 当前段落序号和预算（参考用，非强制）

## 输出

输出当前段落的正文内容。不要输出任何说明、分析或额外内容。只输出该段的正文。

## 当前段落要求

请生成第 [段落编号] 段：

```
段落编号: [X]
段落功能: [功能描述]
预算字数: [Y] 字（参考值，允许 ±20%）
情绪: [主导情绪]
内容类型: [对话/叙述/描写/心理/闲笔]
```

## v5.0 创作约束（按优先级排列）

### 第一优先级：反检测硬约束（必须满足）

1. **段落长度剧烈波动**：
   - 如果上段是长段（>500字），本段必须是短段（<200字）
   - 如果上段是短段（<100字），本段可以是超长段（>800字）
   - 目标：全文最长段 / 最短段 >= 3

2. **极端突发性**：
   - 每段必须包含至少 1 个 1-5 字超短句（"哦。""没。""完了。"）
   - 每段必须包含至少 1 个 50+ 字超长句
   - 短句总量占比 >= 40%
   - 相邻句长度差 >= 8 字的比例 >= 60%

3. **对话碎片化**（如本段含对话）：
   - 至少 30% 的对话必须是：打断、沉默（动作代替）、重复、口吃、答非所问
   - 禁止完整礼貌的对话回合（问→答→回应）

4. **连接词替换**：
   - 绝对禁止：然而、此外、因此、与此同时、最后
   - 替换为：可、不过、还有、于是、那会儿、临了

5. **闲笔插入**（每 2000 字至少 1 处）：
   - 与主线无关的感官细节
   - 必须包含：具体物品 + 气味/声音/触觉 + 一段回忆或判断

6. **视角跳切**（全文至少 1 处）：
   - 第三人称中插入叙述者主观判断（"说来也怪""那年头就这样""平心而论"）

### 第二优先级：原创性硬约束

1. **连续 4 字禁止**：新篇中不得出现与原文任何位置连续 4 字以上相同的片段
2. **零变量残留**：原文的人物名、地名、道具名不得出现
3. **禁止逐句对应**：不得出现新篇第 N 句与原文第 N 句结构相同的情况
4. **骨架相似度 <50%**：事件顺序、段落功能分布必须与原文不同

### 第三优先级：字数约束（放宽）

1. 本段字数参考预算 [Y] 字，允许 ±20%
2. 全文总字数控制在目标 ±5% 内即可
3. 段落级字数偏差由 humanize.py 后处理修正

### 第四优先级：风格约束

1. 模仿骨架卡中的"写法规则"（而非原文具体句子）
2. 情绪曲线走势可以与原文不同（允许情绪错位）
3. 保持语言质感（口语化程度、修辞密度等级）

## 生成策略：多温度拼接（如使用 LLM API）

如果支持参数调节：
- **开头句子**：temperature=0.7（稳定入口）
- **中间句子**：temperature=1.2~1.5（混乱核心区）
- **结尾句子**：temperature=1.0（平衡收束）
- **过渡处**：故意制造语义跳跃

## 自检清单

写完后检查：
- [ ] 本段是否有 1-5 字超短句？
- [ ] 本段是否有 50+ 字超长句？
- [ ] 对话是否碎片化（打断/沉默/重复）？
- [ ] 是否使用了"然而""此外""因此""与此同时""最后"？（有→替换）
- [ ] 是否有与原文连续 4 字相同？（有→重写）
- [ ] 是否与原文逐句对应？（是→重写）
- [ ] 本段字数是否在预算 ±20% 内？

## 输出示例

**正确写法（v5.0 标准）**：

> 相馆里的钟停了。停在四点二十。李默没修它——时间在这间屋子里早已不算数了。点名册搁在柜台的玻璃下面，纸页发脆，像碰一下就会碎成灰，她每天开门前都会数一遍，四十二个名字，一个不多一个不少，今天左边那页缺了一角，不是撕的，像是被什么洇湿过又干透了字就没了，门外的巷子安静得过分，卖馄饨的张婶推车经过轮子碾过石板的声音从窗缝里钻进来又远去了，李默把点名册合上，又打开，还是四十一。说来也怪，她居然松了口气。

检查：
- 超短句："相馆里的钟停了。""停在四点二十。""李默没修它——" ✅
- 超长句："点名册搁在……字就没了"（52字） ✅
- 碎片化：无对话但有意外的情绪转折 ✅
- 无禁止连接词 ✅
- 视角跳切："说来也怪" ✅
