# 02-variable-residue：变量残留扫描

## 1. 什么是变量残留

变量残留是指新篇中仍然保留了原文中的具体人物名、地名、道具名、台词、意象等关节点。这是换元仿写中最危险的原创性问题。

**目标**：变量残留率 = 0%。

## 2. 需要扫描的变量类型

### 2.1 人物类

- 人物姓名（包括姓氏、昵称、外号）
- 人物外貌特征中的特定描述
- 人物职业名称（如果具有特殊性）
- 人物关系称谓（如"王局长""李主任"）

### 2.2 地点类

- 具体地名（城市、街道、建筑名）
- 特定空间名称（如"303 号房""老邮局"）
- 时代背景词（如"1987 年""民国二十三年"）

### 2.3 事件类

- 特定事件名称
- 专有名词（如案件名、运动名）
- 特殊时间词（如"三年前的那个雨夜"）

### 2.4 道具/意象类

- 关键道具名（如"旧怀表""红丝巾"）
- 特定意象（如"墙上的裂缝""窗台上的猫"）
- 标志性物象（如"半枚玉佩""烧焦的纸条"）

### 2.5 台词/语言类

- 原文关键台词（包括 8 字以上相同表达）
- 人物口头禅
- 特定句式或短语

### 2.6 其他

- 特殊数字、价格、日期
- 特定机构名、店铺名
- 原文中的特殊标点组合（如连续破折号用法）
- 原文中反复出现的特定动词（如"摩挲""攥"）—— 如果过度使用，也可能算残留

## 4. 残留处理策略

### 4.1 完全替换

对于明确残留的人物名、地名、道具名，直接替换为新变量。

### 4.2 语义改写

对于残留的句式或短语，重新表达相同意思但用不同词汇和结构。

**原文**：
> 他攥着那只旧怀表，指节发白。

**新篇残留**：
> 他攥着那本点名册，指节发白。

**问题**："指节发白"可能是原文残留。

**修正**：
> 他手指捏着点名册，关节泛白。

### 4.3 结构改写

对于残留的句子结构，改变句式。

**原文**：
> 雨下了一整夜。他坐在窗边。

**新篇残留**：
> 雪下了一整夜。他坐在柜台后面。

**问题**：句式完全复制。

**修正**：
> 雪下了一夜。柜台后的灯还亮着。

## 6. 容易遗漏的残留

### 6.1 变体形式

- 原文："张三" → 新篇残留："老张"
- 原文："北京" → 新篇残留："京城"
- 原文："旧怀表" → 新篇残留："那只怀表"

这些变体形式也需要扫描。

### 6.2 间接引用

- 新篇中描述了与原文完全相同的场景，只是换了个地点名
- 新篇中使用了与原文相同的动作序列，只是换了主语

这些属于"隐性残留"，需要人工复核。

### 6.3 习惯性表达

如果原文作者有某些习惯性表达（如"他深吸一口气"），新篇中频繁使用类似表达，也可能被识别为风格模仿过度。但这不是变量残留，而是风格 DNA 的保留。

## 7. 扫描工具实现

```python
def residue_scan(original_text, new_text, variable_list):
    """变量残留扫描"""
    residues = []
    
    # 精确匹配
    for var in variable_list:
        if var in new_text:
            residues.append({"type": "exact", "content": var})
    
    # 8 字以上公共子串
    lcs = longest_common_substring(original_text, new_text)
    if len(lcs) >= 8:
        residues.append({"type": "lcs", "content": lcs})
    
    # 变体扫描（如"老张"对应"张三"）
    for var in variable_list:
        variants = generate_variants(var)
        for variant in variants:
            if variant in new_text:
                residues.append({"type": "variant", "content": variant})
    
    return residues
```
