# knowledge/net-style/01-genres
> 所属类别：网络文学风格 | 关联文件：02-pleasure-mechanism, 03-world-building, 04-characters | 最后更新自搜索批次：A1_1–A1_7

## 核心定义与概念边界

网络文学流派是指基于题材背景、核心设定与读者期待而形成的类型分类体系。它不同于传统文学的"主义"划分（如现实主义、浪漫主义），而是以**故事性背景**和**快感类型**为分类依据。起点中文网等平台将其划分为玄幻、奇幻、武侠、仙侠、都市、现实、军事、历史、游戏、体育、科幻、灵异、女生网、二次元等十余大门类。

**它不是什么：**
- 不是文学思潮或美学运动——网文流派不涉及创作理念的宣言或论战，而是市场与读者需求驱动的类型分化
- 不是严格的学术分类——同一作品可横跨多个流派标签（如《诡秘之主》同时是玄幻+克苏鲁+序列升级），流派边界具有高度渗透性
- 不是固定不变的——新流派持续涌现（如2020年代兴起的基建文、马甲文），旧流派可能融合或消亡

**与相邻概念的区别：**
- 流派 vs 题材：题材是故事内容的表层描述（如"穿越到唐朝"），流派是包含快感期待、叙事惯例和读者契约的深层结构
- 流派 vs 套路：套路是流派内部的叙事模式（如"扮猪吃虎"是玄幻流的套路），流派是套路的容器和语境
- 流派 vs 频道：频道是平台运营的分类标签，流派是读者与作者共同认可的类型共识——两者有重叠但不等同

## 内部分歧与流派

### 一、主流流派体系

**玄幻类**——网文第一大类型，不受科学与人文限制，以想象力自由发挥为核心特征。内部细分：
- **东方玄幻**：由罗森《风姿物语》开创，以原创世界观为舞台，融合中国古代风水命理、修真体系，是奇幻修真流的前身之一
- **异界大陆**：以完全架空的异世界为背景，主角穿越或重生至异界，以力量升级为主线
- **异世界穿越**：现代主角穿越至异世界，利用现代知识或特殊能力获得优势

**仙侠类**——脱胎于武侠与神魔小说，以剑仙侠客为核心人物，以修仙练道为核心情节。内部细分：
- **修真流**：以萧潜《缥缈之旅》为开山鼻祖，建立了一套完整的修炼等级体系（炼气→筑基→金丹→元婴→化神→……），强调修炼过程的系统性与等级感
- **凡人流**：以忘语《凡人修仙传》为鼻祖，主角资质平庸，依靠机缘与算计在修仙世界求存，核心爽点从"天赋碾压"转向"凡人逆袭"
- **古典仙侠**：更接近传统武侠与神魔小说的美学，强调仙风道骨与侠义精神

**都市类**——以现代都市为背景，内部细分：
- **都市异能**：现代都市中存在超自然力量
- **都市重生**：主角重生回到过去，利用先知优势改变命运
- **都市职场/商战**：以现实职场或商业竞争为核心冲突

**历史类**——以历史为背景，内部细分：
- **穿越历史**：现代人穿越至古代，利用现代知识改变历史
- **架空历史**：以历史为蓝本的完全架空世界
- **历史考据**：近年兴起的"老白文"类型，强调历史细节的考据与推演

**科幻类**——网络科幻小说经历了从"野蛮生长"的边缘化状态到迈上文学接受中心舞台的历程，内部细分包括硬科幻、软科幻、末世流等

**游戏类**——以游戏世界为背景或核心设定，包括虚拟现实、电竞、游戏穿越等

### 二、特色流派

**无限流**——由zhttty《无限恐怖》开创，核心框架为"主神空间+副本闯关+团队进化"。主角被召唤至主神空间，在不同影视/动漫/游戏副本中执行任务，获得积分强化自身。关键特征：团战机制、副本切换、生存压力

**诸天流**——与无限流的双胞胎流派，核心区别在于：诸天流没有主神空间的约束，主角独自穿梭不同世界，目标自主设定，收益来自过程获得。诸天流更注重个体成长与自由探索，无限流更强调团队协作与生存竞争

**系统流**——主角获得某种"系统"辅助（如签到系统、模拟器、聊天群），系统提供任务、奖励与成长路径。系统流与无限流的相似度高于诸天流——都有外部规则约束与奖励机制

**种田文/基建文**——以经营建设为核心快感，主角从零开始建设领地/国家/基地。种田文偏日常温馨，基建文偏宏大建设

**年代文**——以特定历史年代（多为六七十年代）为背景，融合穿越/重生元素，核心爽点在于利用先知优势在物质匮乏年代发家致富

**马甲文**——主角拥有多重身份（马甲），核心爽点在于身份揭露时的震惊效果

**甜宠文/虐文**——言情类两大情绪策略流派。甜宠文全程高甜无虐，虐文以情感折磨为核心驱动力

**克苏鲁网文**——洛夫克拉夫特式恐怖的本土化改编，以爱潜水的乌贼《诡秘之主》为代表作，将克苏鲁的不可名状之恐惧与网文升级体系融合

**轻小说本土化/二次元网文**——受日式轻小说影响的网文类型，特征包括：轻松幽默的叙事基调、二次元文化梗、角色萌属性标签化

### 三、流派交叉融合现象

当代网文最显著的趋势是**流派融合**——单一流派标签已无法准确描述作品。例如：
- 《诡秘之主》= 克苏鲁 + 维多利亚 + 序列升级 + 职业体系
- 《我有一座冒险屋》= 恐怖 + 系统 + 都市 + 升级
- 《大奉打更人》= 历史 + 修仙 + 探案 + 爽文

这种融合不是简单的标签叠加，而是在叙事逻辑层面的深度整合——不同流派的快感机制被编织进同一叙事框架。

## 机制深解

### 流派生成的动力学

网文流派的生成遵循"开创者→模仿者→固化→变异"的循环：
1. **开创期**：某部作品创造了新的叙事模式（如《无限恐怖》创造无限流）
2. **繁荣期**：大量模仿者涌入，形成类型共识与读者期待
3. **固化期**：套路被穷尽，读者产生审美疲劳
4. **变异期**：作者在原有框架上创新，或与其他流派融合产生新变体

### 流派与读者契约

每个流派都隐含着作者与读者之间的**类型契约**——读者选择某流派时，已经携带了特定的期待：
- 玄幻读者期待：力量升级、越级挑战、宏大世界观
- 言情读者期待：情感发展、人物互动、情感高潮
- 无限流读者期待：副本多样性、团队博弈、生存压力

违反类型契约（如玄幻文突然变成纯言情）会导致读者流失，但巧妙地**延迟满足**契约（如先压抑后爆发）则是高阶技巧。

### 平台分类与流派生态

起点中文网等平台的分类标签直接影响了流派的生态——平台推荐算法基于标签匹配，使得标签选择成为作品曝光的关键策略。这导致了一种"标签博弈"现象：作者可能选择流量更大的标签而非最准确的标签。

## 案例库

### 案例1：《凡人修仙传》（忘语）——凡人流的开创
韩立资质平庸，没有天赋碾压，依靠谨慎算计和机缘巧合在修仙世界求存。开创了"凡人也能修仙"的叙事范式，核心创新在于将爽点从"天赋碾压"转向"凡人逆袭"，使读者代入感大幅提升。

### 案例2：《无限恐怖》（zhttty）——无限流的开创
郑吒被主神空间召唤，在《生化危机》《异形》等副本中执行任务。开创了"主神空间+副本闯关+团队进化"的核心框架，后续所有无限流作品本质上都在此框架上创新。

### 案例3：《诡秘之主》（爱潜水的乌贼）——克苏鲁网文的巅峰
融合克苏鲁神话、维多利亚时代、序列途径体系，创造了"扮演法"升级机制。将不可名状之恐惧与网文升级爽感融合，是流派融合的典范。

### 案例4：《缥缈之旅》（萧潜）——修真流的开山
建立了完整的修炼等级体系（炼气→筑基→金丹→元婴→化神……），这套体系成为后世修真/仙侠文的通用设定模板。

### 案例5：《寻秦记》（黄易）——穿越流鼻祖
项少龙穿越至战国时代，利用现代知识与先知优势在历史洪流中求存。开创了"现代人穿越古代"的叙事范式。

### 案例6：《风姿物语》（罗森）——东方玄幻开创
以原创世界观为舞台，融合中国古代风水命理，开创了东方玄幻这一细分流派。

## 实践操作指南

### 流派选择决策树
1. 确定核心爽点类型（升级/情感/探索/经营/恐怖）
2. 匹配最契合的流派标签
3. 评估该流派的竞争密度（红海vs蓝海）
4. 考虑流派融合的可能性——在热门流派中加入冷门元素形成差异化

### 流派融合操作方法
- **主+辅模式**：选择一个主流派承载核心叙事，一个辅助流派提供特色元素
- **双核模式**：两个流派各占50%权重，需要找到两者的叙事接口
- **嵌套模式**：外层是一个流派框架，内层嵌套另一个流派的玩法

### 类型契约自检清单
- [ ] 读者选择我的流派标签时，期待什么？
- [ ] 我是否在开篇就建立了类型预期？
- [ ] 我是否在满足契约的同时提供了意外惊喜？
- [ ] 我是否违反了核心契约而不自知？

## 常见误区与反模式

1. **标签堆砌**：给作品贴上过多流派标签，导致类型预期混乱，读者无法建立稳定的阅读期待
2. **违反类型契约不自知**：在玄幻文中突然插入大量日常种田内容，导致爽感断裂
3. **盲目追热点**：选择竞争过密的热门流派，作品淹没在同质化海洋中
4. **忽视流派演变**：使用已过时的流派套路（如2010年代的"废柴逆袭"模板），读者已产生免疫力
5. **流派融合无接口**：两个流派简单并列而不在叙事逻辑层面整合，导致阅读体验割裂
